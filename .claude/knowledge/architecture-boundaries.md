# Architecture & System Boundaries

- Corne keymap bindings are ordered left half then right half for each alpha row, followed by thumbs: left hand positions `0-5`, `12-17`, `24-29`; right hand positions `6-11`, `18-23`, `30-35`; thumbs `36-41`. Use these indices for positional hold-tap settings such as `hold-trigger-key-positions`.
